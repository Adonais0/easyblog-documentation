# Customization

EasyBlog offers several customization options you can try to make your blog site look different. {{ className: 'lead' }}

```tsx
<NextBlogListPage 
  ...
  type="grid" // grid | list
  darkMode={true} // true | false, default true
  displayOptions={{ // Customize based on your need
  showThumbnail: true,
  showReadingTime: true,
  showExcerpt: true,
  showTags: true,
  showDate: true,
  showAuthor: true,
  showCategory: true,
}}
/>
```
